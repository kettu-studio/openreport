# openreport CLI

CLI para subir los reportes de trivy y gitleaks a OpenReport

### Instalación y uso

Para usar esta herramienta, simplemente descárgala y hazla ejecutable:

1. **Descarga el script:**
   ```bash
   curl -L -o openreport https://raw.githubusercontent.com/kettu-studio/openreport/main/cli/openreport && chmod +x openreport && sudo mv openreport /usr/local/bin/
